LMS

Learning management system

Moodle

Modular Object-Oriented Dynamic Learning Environment. Представляет собой свободное веб-приложение, предоставляющее возможность создавать сайты для онлайн-обучения.

eTutorium

Облачная LMS со встроенной вебинарной платформой, которая позволяет решать eLearning задачи. Включает в себя редактор курсов и тестов, диалоговый тренажер.

Open edX

Платформа с открытым исходным кодом, которая поддерживает курсы edX. Учреждения могут размещать свои собственные экземпляры Open edX и предлагать свои собственные курсы.

Agylia

Мобильная игибкая LMS, которая позволяет учащимся получать доступ к цифровому обучению на своих предпочтительных устройствах.

iSpring Learn

Облачная система управления обучением для обучения и оценки сотрудников или студентов в Интернете. Российская разработка.

Bolt Spark LMS

Высокозащищенная, облачная LMS с функциями аналитики, встроенного перевода, включающая элементы геймификации, управление траекторией и прогрессом.

Сервисы

Плюсы и минусы

CourseLab

Инструмент для создания электронных курсов и
интерактивных учебных материалов.
  • Большая библиотека элементов, можно создавать курсы с геймификацией и нелинейным сценарием.
  • Подходит для быстрого создания тренажеров — симуляторов программного обеспечения.
  • Гибкие настройки и возможность добавлять свои функции при помощи программирования
  • Отсутствие необходимых обучающих материалов в свободном доступе
  • Работа с действиями крайне неудобная. Добавляя на слайд новый объект, над ним приходится поработать, отключая все декоративные надстройки.
  • Жалобы на частое зависание окна настройк.
  • Не всегда работают программируемые действия.
  • Сложность в программировании объектов.
  • Визуально устаревший интерфейс как самой программы, так и стандартных шаблонов и объектов, из которых состоят слайды.

Веб-сайт

JustClick

Сервис конструирования и продажи
онлайн-курсов

  • Есть рассылка, конструктор сайтов, система приема платежей, кабинет ученика, система сквозной веб-аналитики.
  • Адаптирован под мобильные устройства.
  • Своя партнерская программа.
  • Поддержка различных способов оплаты.
  • Нет дополнительной платы за количество активных учеников.
  • Для освоения сайта есть бесплатный режим.
  • Большой выбор шаблонов сайта и удобный визуальный редактор страниц. Легко настроить без программиста.
  • Объем хранения данных совсем небольшой – максимальный пакет, который указан на сайте, составляет всего 3 Гб.
  • Частые обновления и изменения правил сервиса.
  • Много негатива о компании в интернете. Особенно о команде поддержки и возможностях самого сервисе.

Веб-сайт

Teachable

Платформа для создания и продажи онлайн-курсов

  • Собственная партнерская программа.
  • Безлимитный хостинг видео, неограниченное количество учеников и курсов.
  • Возможность перевода на различные языки, в том числе русский.
  • Встроенные инструменты создания отчетов по курсам.
  • Оптимизация под различные браузеры, мобильные устройства.
  • Интерфейс англоязычный.
  • Нет встроенных возможностей геймификации.
  • Нет возможности выдавать сертификаты студентам.
  • Нет функции управления группами.
  • Нет вебинаров, только создание онлайн-уроков.

Веб-сайт

GetCourse

Сервис для онлайн-обучения и проведения вебинаров

  • Автоматизация бизнес-процессов. напоминания о предстоящем вебинаре, проверка тестов и выдача пользователям сертификатов.
  • Интеграция с 13 сторонними сервисами. Есть интеграция с Youtube и другими сервисами для проведения вебинаров.
  • Встроенный механизм рассылок.
  • Конструктор страниц — с его помощью можно самостоятельно собрать адаптивную страницу.
  • Формы продажи, офферы, продукты, статистика, полноценная CRM.
  • Своеобразный интерфейс управления.
  • Хостинг видео оплачивается отдельно.
  • Нет DKIM-подписи, а это значит, что письма могут попадать в спам.
  • Много жалоб на поддержку клиентов.
  • Много дополнительных услуг, которые нужно оплачивать.

Веб-сайт

Gurucan

Платформа для создания онлайн-курсов
Упор делают на онлайн-курсы в форме мобильных приложений.

  • Кастомизация продукта: возможность сделать свой дизайн, подключить свой собственный домен, персональные настройки домена.
  • Безлимитный 4К хостинг видео.
  • Своя партнерская программа. Предлагается 30% с каждого приведенного клиента, который пользуется сервисом.
  • Создание настоящего мобильного приложения для онлайн-школы. Поддерживаются операционные системы iOS и Android.
  • Отзывчивый чат поддержки.
  • Есть своя база знаний.
  • Геймификация только в стадии разработки.
  • Нет возможности сделать сертификат о прохождении курса.
  • Новый сервис, поэтому пока немного отзывов и информации в Интернете.

CMS

Инструмены для CMS.

LMS Lifter

Бесплатный плагин WordPress, который подразумевает собой систему управления обучением для создания курсов и прохождения их в режиме онлайн. С его помощью вы создадите курсы разного типа, сможете добавить тесты, а также вознаграждения за достижение определенного результата и сертификаты, которые получат студенты по прохождению ваших курсов.

Quiz Deluxe Expert

Профессиональный компонент для создания тестов или онлайн опросов любой сложности и с любой логикой подсчета результатов. Широко используется в дистанционном обучении, тестировании персонала и отборе кандидатов, проведения экзаменов, олимпиад и организации учебного процесса.

LearnPress

Комплексный плагин для WordPress, который можно использовать для создания и продажи онлайн-курсов. Каждый учебный план курса может быть составлен с уроками и викторинами, которыми можно управлять с помощью простого в использовании пользовательского интерфейса.

iJoomla Guru PRO

Компонент для создания интернет школы с классами и курсами. Есть возможность сосоздания различных викторин, медиа библиотек, форума обучающихся и системы комментариев. Онлайн продажа направлений осуществляется через разные тарифные планы при помощи встроенных плагинов оплаты.

Вывод

Системы обучения отличаются не только функционалом, но и тем, какие проблемы они могут решить. Поэтому универсального решения нет. Каждый инструмент отвечает конкретным целям: корпоративное обучение, продажа курсов, дистанционное обучение в вузах. Чтобы понять, подойдет вам система или нет, надо попробовать решения каждого провайдера.
